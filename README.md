## Feature

- 扩展 Internlm2-7B 的能力，接入音乐生成功能
- 更高质量的音乐知识、自我认知等数据
- 通用的 ABC 乐谱模板匹配
- 支持 ABC 乐谱格式转换 (wav, midi, etc.)
